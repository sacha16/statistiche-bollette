# Statistiche e Confronto Bollette

Questo repository contiene un'applicazione web statica per l'analisi e il confronto delle bollette di luce e gas tra diversi fornitori (ad esempio, Eni Plenitude e Octopus Energy).

## Avvio in Locale

L'applicazione è configurata con **Vite** come server di sviluppo locale.

### Requisiti
- **Node.js** (versione consigliata 18+)

### Istruzioni per l'Avvio

1. Installa le dipendenze:
   ```bash
   npm install
   ```

2. Avvia il server di sviluppo locale:
   ```bash
   npm run dev
   ```

Il server si avvierà in locale ed aprirà automaticamente l'applicazione nel tuo browser predefinito all'indirizzo standard (solitamente `http://localhost:5173`).
